# AI Appointment Setter — Spec

## Overview
Premium SaaS startup website for an AI-powered appointment setting product. Converts visitors into demo bookings and free trial signups.

## Tech Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS + glassmorphism utilities
- Framer Motion for animations
- Lucide React for icons
- Inter font (Google Fonts)

## Color Palette
- Primary: #6C63FF
- Secondary: #111827
- Accent: #00D4FF
- Background: #0B1020
- Surface: rgba(255,255,255,0.05)
- Text Primary: #FFFFFF
- Text Secondary: #9CA3AF

## Key Decisions
- Dark mode is default, light mode toggled
- Mobile-first responsive design
- All sections on single landing page
- API routes for contact form + newsletter
- SEO optimized with JSON-LD schema
